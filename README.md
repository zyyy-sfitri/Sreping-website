# Scraping Website - Analisis Sentimen Ulasan Toko G2G

Proyek ini bertujuan untuk melakukan **scraping data ulasan produk** dari Tokopedia dan menganalisis **sentimen pengguna** terhadap produk tersebut.

## Fitur
- Mengambil ulasan produk dari Tokopedia secara otomatis.
- Membersihkan data ulasan (data cleansing) untuk menghilangkan noise seperti:
  - Tanda baca
  - Emoji
  - Stopwords
- Menganalisis sentimen ulasan menjadi:
  - Positif
  - Negatif
  - Netral
